# বিজ্ঞানচিন্তায় প্রকাশিত



২০২৫ সনে বিজ্ঞানচিন্তার সাথে যোগাযোগ হয় এবং সেখানে নিয়মিত কিছু লেখা প্রকাশিত করি। লেখাগুলিকে আরেক গুণগত মান ভালো করার জন্য বিজ্ঞানচিন্তা'র সম্পাদক মন্ডলির কাছে কৃতজ্ঞ।&#x20;
